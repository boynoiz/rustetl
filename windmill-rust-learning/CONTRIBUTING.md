# Contributing

This is a personal learning repository, but contributions are welcome!

## How to Contribute

### Adding New Examples

1. Create script in appropriate directory:
   - `windmill-scripts/01-basics/` - Basic examples
   - `windmill-scripts/02-advanced/` - Advanced features
   - `windmill-scripts/03-parameterized/` - With parameters
   - `windmill-scripts/04-database/` - Database operations

2. Test locally in `local-testing/`

3. Document in README.md within that directory

### Improving Documentation

- Fix typos
- Add clarifications
- Share your learnings
- Add troubleshooting tips

### Sharing Workflows

If you build something cool:
- Add to appropriate section
- Document use case
- Include performance notes
- Share lessons learned

## Code Style

- Follow Rust conventions (`cargo fmt`)
- Use `clippy` for linting
- Add comments for complex logic
- Keep examples simple and focused

## Commit Messages

Use conventional commits:
- `feat: add new example for X`
- `docs: improve README for Y`
- `fix: correct error in Z`
- `refactor: reorganize scripts`

## Questions?

Open an issue or discussion!
